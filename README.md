# 📱 Mobile Recharge System (Java)

A simple Java application that simulates a mobile phone recharge system, allowing users to select predefined recharge amounts.

---

## 🚀 Features

- 📲 User input for phone number  
- 💰 Selection of recharge amount  
- 🔢 Validation of predefined recharge values  
- 📢 Feedback messages for successful or invalid operations  

---

## 🧠 Concepts Applied

- Java fundamentals  
- User input with `Scanner`  
- Conditional logic (`switch-case`)  
- Arrays  
- Basic input validation
